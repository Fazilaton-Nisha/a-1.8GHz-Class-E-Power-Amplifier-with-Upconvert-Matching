# a-1.8GHz-Class-E-Power-Amplifier-with-Upconvert-Matching
A 1.8 GHz Class-E power amplifier with 2.5 V supply using ATF-34143 E-pHEMT transistor, achieving 71.3% efficiency and 23.13 dB gain. Optimized beyond theory to address parasitics via Keysight ADS simulations. Suitable for low-power IoT and wireless systems. Tech: ADS, Class-E, RF design, impedance matching
